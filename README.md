OBJETIVO:

Este sistema foi criado para facilitar o cadastro e a consulta de alunos. Ele permite registrar informações básicas, localizar um aluno rapidamente e manter esses dados salvos para uso futuro.

Com ele, é possível:

- Cadastrar novos alunos
- Buscar alunos pelo nome ou matrícula
- Atualizar informações quando necessário
- Remover alunos que não fazem mais parte da lista

Todas as informações ficam guardadas automaticamente em um arquivo CSV, garantindo que nada seja perdido. O sistema funciona por meio de um menu simples, que orienta o usuário em cada etapa.
